# Role-Based User Management System

A full-stack application with an ASP.NET Core Web API backend and a Flutter frontend, implementing JWT authentication and role-based access control.

## Presentation Video
Check out the project demonstration here: https://youtu.be/6amRd955rxI

## Tech Stack
- **Backend**: ASP.NET Core 9.0, Entity Framework Core, SQLite, JWT.
- **Frontend**: Flutter, Provider, JWT Decoder.

## Features
- **Admin**: Manage users, roles, assign roles, edit profile, change password.
- **User**: View info, edit profile, change password, delete account.
